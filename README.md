# Projeto Exemplo de API para cadastrar Projetos e Tarefas ao Projeto

## Requerimento
  * Node 10.14.1
  * Yarn ou NPM

## Dependências
  * Express
  * Nodemon
  * Sucrase
  
## Execução:
#
```
$ yarn dev
```

O projecto ficará disponível http://localhost:9090/ ou em outra porta caso mude o documento [server.js]
